# 天翼视联助手 Chrome 插件

一款专为天翼视联平台用户设计的 Chrome 浏览器插件，提供多种实用功能，帮助用户更高效地管理设备和执行操作。

## 📋 功能特性

### 1. 用户信息页面
- 显示 Cookie (Python 格式)，方便复制使用
- 显示用户 ID (userId)
- 显示手机号 (account)
- 一键刷新数据

### 2. 批量级联功能
- 支持手动输入设备编码（多个用逗号分隔）
- 支持上传 CSV 文件批量导入设备编码
- 提供 CSV 模板下载
- 批量创建级联任务
- 支持导出级联清单
- 一键清除数据

### 3. 批量设备查询
- 支持手动输入设备编码（多个用逗号分隔）
- 支持上传 CSV 文件批量导入设备编码
- 提供 CSV 模板下载
- 批量查询设备在线状态
- 支持导出查询结果
- 一键清除数据

### 4. 设备清单查询
- 输入手机号获取企业主列表
- 选择企业主后查询设备清单
- 支持分页获取所有设备数据
- 表格形式展示设备信息（序号、设备名称、设备编码、设备型号、目录）
- 支持导出设备清单
- 一键清除数据
- 显示查询进度条
- 支持中途停止查询

## 📦 安装方法

1. **下载插件**：将插件代码下载到本地
2. **打开 Chrome 扩展管理**：在 Chrome 浏览器中输入 `chrome://extensions/`
3. **开启开发者模式**：点击右上角的 "开发者模式" 开关
4. **加载已解压的扩展程序**：点击 "加载已解压的扩展程序"，选择插件所在目录
5. **完成安装**：插件会自动加载并显示在浏览器工具栏

## 🚀 使用说明

### 打开插件
- 点击浏览器工具栏中的插件图标，或
- 使用快捷键 `Ctrl+Shift+Y` (Windows) 或 `Cmd+Shift+Y` (Mac)
- 也可以通过浏览器右侧边栏打开（需要在扩展管理中启用边栏功能）
